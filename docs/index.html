RNA-seq using Nextflow

Synopsis:

Introduces downstream analysis steps of RNA sequencing analysis using R/Rstudio - including how to analyse raw count data to obtain differentially expressed genes and pathways. This will be performed on Pawsey Nimbus Cloud.

Target audience:

Researchers, students and life scientists who are new to RNA sequencing for differential expression and pathway analysis. The course is beginner friendly and intended for those interested in using R/Rstudio for their analysis.

Interesting resources:

  • Introduction to Unix/Linux command line
  • Introduction to R/RStudio

Follow-on courses:

This course is part two of the two-part RNA sequencing training series.

We recommend attending both courses if you would like an end-to-end overview of how to perform the analysis. If you are only interested in the technologies used, you may be able to get away with attending only the parts of interest :smiley:.
